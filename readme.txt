应用：

软件主要应用于抗体序列的比对及抗体CDR/FR区域的标注。


序列比对：

输入两个氨基酸序列后，点击 "开始比对"，结果会显示在上方的滚动文本框中。
缺口（-）会被标记为红色，匹配的字符会被标记为蓝色。
CDR/Fr 标注：
输入抗体序列后，选择标注规则（如 Kabat），点击 "标注CDR/FR"。
CDR 区域会被标记为黄色背景，FR 区域会被标记为浅蓝色背景。
